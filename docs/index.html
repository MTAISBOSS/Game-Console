Pico Tetris Game Console

A classic Tetris implementation for Raspberry Pi Pico with ILI9341 display

Game Preview

Experience classic Tetris with colorful blocks and smooth gameplay on your Raspberry Pi Pico.

Hardware Setup

Compact setup with ILI9341 display and tactile buttons for full game control.

Project Features

Classic Gameplay

Faithful recreation of the original Tetris mechanics with all 7 tetromino pieces.

Colorful Display

Vibrant colored blocks using custom BMP files for each tetromino type.

Background Music

Integrated melody player that plays the classic Tetris theme during gameplay.

Responsive Controls

Debounced button inputs for precise piece movement and rotation.

Hardware Components

  • Raspberry Pi Pico
  • ILI9341 TFT Display (240x320)
  • Tactile buttons (x4)
  • Breadboard and jumper wires
  • SPI interface for display
  • Digital inputs for buttons
  • 5V power supply

Technical Implementation

The game is built using CircuitPython with displayio for graphics rendering. Key components include:

Display Initialization

# Initialize SPI
spi = busio.SPI(clock=clk_pin, MOSI=mosi_pin)

# Initialize the display bus
display_bus = displayio.FourWire(spi, command=dc_pin, chip_select=cs_pin, reset=reset_pin)

# Initialize the ILI9341 display (rotation set to 90 for vertical orientation)
display = adafruit_ili9341.ILI9341(display_bus, width=240, height=320, rotation=90)

Game Loop

async def tetris_game():
    while True:
        # Handle controls
        if debouncer_left.debounced_press():
            move_piece_left()
            
        # Game logic
        if not move_piece_down():
            place_piece()
            clear_lines()
            spawn_new_piece()
            
        await asyncio.sleep(0.2)

Project Files

The project consists of several key files:

  • code.py - Main game logic and display handling
  • melody.py - Music player for the Tetris theme
  • block_<color>.bmp - Colored block images for each tetromino
